Quadratic formula calculator solves quadratic equations using the quadratic formula. Quadratic equations are solved for real roots and complex roots.

[https://www.calculator.io/quadratic-formula-calculator/](https://www.calculator.io/quadratic-formula-calculator/)

== Usage ==

Add the Quadratic Formula Calculator shortcode to your page, post or sidebar:

`[ci_quadratic_formula_calculator]`

Add the following code to your template where you would like the Quadratic Formula Calculator to appear:

`<?php display_ci_quadratic_formula_calculator(); ?>`

1. Upload the Quadratic Formula Calculator /ci_quadratic_formula_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Quadratic Formula Calculator plugin through the "Plugins" menu in WordPress.
